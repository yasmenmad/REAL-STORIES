# Real Stories

> The most extraordinary films and series ever made — all based on real people who changed history, broke rules, and refused to be forgotten.

---

## Features

- 85+ titles across 8 categories — Shocking, Music, Science, Crime, War, Royals, Sports, Politics
- Live movie posters fetched in real-time via TMDB API
- AI-powered search — describe what you want in natural language (Groq / LLaMA)
- AI Chat — ask anything about a film or the real story behind it
- AI Deep Dive — get a literary, critic-level breakdown of any title
- Mood Recommender — pick a vibe, get curated picks instantly
- Custom cursor and cinematic dark UI with gold accents

---

## Tech Stack

| Layer      | Technology                        |
|------------|-----------------------------------|
| Frontend   | Vanilla HTML, CSS, JavaScript     |
| AI         | Groq API (LLaMA 3.1)              |
| Posters    | TMDB API                          |
| Fonts      | Google Fonts — Cormorant Garamond + Archivo |

---

## How to Run

No installation needed. Just open the file:

    true-stories-v2.html

Drag and drop the HTML file into any browser — it works instantly.
